Lambda Substitution... Literally
================================

A toy plugin for Sublime Text 3 that allows you to use nice Unicode `位` symbols in your code and text.

Functionality
-------------
- Remaps the `\` key so it outputs `位` characters (press `\` twice for a backslash).
- On saving a file, `位` characters will be optionally converted to backslashes.

Usage
-----
- Press `\`.
- Edit `Preferences.sublime-settings` if you want Unicode `位` characters to be saved as they are.
